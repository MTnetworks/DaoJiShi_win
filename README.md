# 会议演讲倒计时软件 (Conference Speech Countdown Timer)

## 简介
一款专为会议、演讲、比赛及各种需要时间管理的场合设计的倒计时工具。它具备简洁现代的界面、强大的自定义功能和稳定的性能，能够帮助演讲者精确把控时间，同时支持透明背景悬浮显示，完美配合 PPT 演示使用。

## 核心功能

### 1. 多模式显示
- **普通模式**：包含完整的控制面板，显示预设、进度条和控制按钮，适合设置和调试。
- **精简模式（PPT 模式）**：倒计时开始后自动进入，隐藏所有干扰元素（按钮、标题栏），背景支持透明，仅保留清晰的时间数字，可悬浮在 PPT 或其他软件上方。
- **智能记忆**：软件会自动记忆普通模式和精简模式下窗口的大小和位置，切换时无缝衔接。

### 2. 高度自定义外观
- **主题切换**：内置“明亮”和“深色”两种主题，适应不同环境光线。
- **字体调节**：支持 20px - 500px 超大范围的字体大小调节，满足不同距离的观看需求。
- **透明度控制**：
  - **背景透明**：支持仅背景透明而文字不透明，确保文字清晰可见。
  - **动态调整**：可通过设置或快捷键实时调整背景透明度。
- **显示格式**：
  - **分:秒** (例如 05:00)
  - **纯秒数** (例如 300)
  - **百分比** (例如 100.0% -> 0.0%)

### 3. 灵活的预设管理
- **多组预设**：内置 5分钟、10分钟、15分钟 等常用预设。
- **自定义编辑**：用户可自由添加、删除、修改预设名称、时长和提醒时间。
- **导入导出**：支持将所有配置（包括预设、快捷键、主题设置）导出为 JSON 文件，方便在不同设备间同步或备份。

### 4. 全方位的提醒机制
- **视觉提醒**：
  - **闪烁提示**：当倒计时结束或触发预警时，时间背景会闪烁（颜色可自定义，默认为红色）。
  - **进度条**：直观展示剩余时间的百分比进度。
- **听觉提醒**：
  - **背景音乐**：支持为每个预设绑定独立的音频文件。
  - **淡入淡出**：音频播放支持自定义时长的淡入效果，避免突兀。
  - **音量控制**：内置独立的音量调节功能。

### 5. 便捷的操作控制
- **全局快捷键**：即使软件在后台运行，也可以通过键盘控制。
  - **F1**: 开始/暂停
  - **F2**: 重置
  - **F3**: 切换下一个预设
  - **F4**: 切换上一个预设
  - **F5**: 显示/隐藏窗口
  - **F6**: 切换窗口置顶
  - **F7/F8**: 增加/减少透明度
  - **Ctrl+M**: 静音
  - *注：所有快捷键均支持用户自定义修改。*
- **窗口控制**：
  - **边缘拖拽**：鼠标悬停在窗口任意边缘或角落即可拖拽调整大小。
  - **窗口置顶**：一键设置窗口始终显示在最顶层，防止被其他应用遮挡。
  - **系统托盘**：支持最小化到系统托盘运行，保持桌面整洁。

## 技术特性
- **高精度计时**：基于系统时间戳的毫秒级计时核心，确保长时间运行无误差。
- **跨平台架构**：基于 Python + PyQt5 开发，具有良好的跨平台兼容性。
- **轻量级**：资源占用低，运行流畅。

## 使用说明
1. **启动**：运行程序，进入主界面。
2. **设置**：点击“设置”按钮，配置预设时间、快捷键、音频和外观选项。
3. **开始**：选择一个预设，点击“开始”或按 F1 键。软件将自动进入精简模式（如果设置了）。
4. **调整**：在倒计时过程中，可直接拖动数字移动窗口，或拖动边缘改变大小。
5. **结束**：倒计时结束后，背景会变为预设的警告色并闪烁，提醒演讲者时间到。

## Windows 打包
执行打包命令：
```bash
pyinstaller DaoJiShi.spec --clean --noconfirm
```

## Linux (UOS/Deepin/Ubuntu) 打包指南

本项目提供了构建 `.deb` 安装包的脚本，方便在 UOS、Deepin 或 Ubuntu 等基于 Debian 的系统上安装。

### 前置条件
确保系统已安装 Python 3 和 pip。
推荐在 UOS 终端中执行以下操作。

### 打包步骤

1. **进入 linux 目录**
   打开终端，切换到项目目录下的 `linux` 文件夹：
   ```bash
   cd linux
   ```

2. **赋予脚本执行权限**
   ```bash
   chmod +x build_deb.sh
   ```

3. **执行打包脚本**
   ```bash
   ./build_deb.sh
   ```
   脚本会自动：
   - 安装依赖 (pyinstaller 等)
   - 编译 Python 代码为可执行文件
   - 生成 `.deb` 安装包

4. **安装**
   打包完成后，会在 `linux` 目录下生成 `.deb` 文件（例如 `daojishi_1.0.0_amd64.deb`）。
   安装命令：
   ```bash
   sudo dpkg -i daojishi_1.0.0_amd64.deb
   ```

### 注意事项
- **快捷键权限**：由于 Linux 的安全机制，全局快捷键（使用 `keyboard` 库）可能需要 root 权限或配置 udev 规则才能生效。如果遇到快捷键不响应，请尝试以 sudo 运行程序，或者在系统设置中手动绑定快捷键。
- **图标**：脚本会尝试使用项目中的 `.ico` 图标。如果系统安装了 `imagemagick`，脚本会自动将其转换为 PNG 格式以获得更好的显示效果。
